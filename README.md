# TTRPG-Server-Manager

Discord bot to automate tasks in my TTRPG group's server.

Hosted on [Cloudflare](https://dash.cloudflare.com/).<br>
Created on [Discord's Dev Portal](https://discord.com/developers/applications).

## Commands

### /summary

Create and pin a summary message for the current channel. Edit the pinned summary message if it already exists.
